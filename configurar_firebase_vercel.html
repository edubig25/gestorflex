Facilitador Firebase → Vercel

Gere a variável FIREBASE_SERVICE_ACCOUNT_BASE64 sem terminal.
Abrir Vercel

1. Baixe a chave no Firebase

1
Entre no Firebase Console e abra o projeto do Gestor Flex.
2
Vá em Configurações do projeto → Contas de serviço.
3
Clique em Gerar nova chave privada e baixe o arquivo JSON.
4
Renomeie o arquivo para serviceAccountKey.json.
Não envie esse JSON para ninguém e não suba no GitHub. Ele é uma chave privada.

2. Converter para Base64

Selecione o arquivo JSON. A conversão acontece somente no seu navegador.

Base64 copiado com sucesso!
Mesmo hospedado na Vercel, este arquivo usa o seletor local do navegador. Ele não envia o JSON para o servidor.

3. Colar na Vercel

Project: gestorflex-zeta
Settings: Environment Variables
Name: FIREBASE_SERVICE_ACCOUNT_BASE64
Value: cole o Base64 copiado
Environment: Production
