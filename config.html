Configurar Firebase na Vercel

Use esta página para gerar a variável FIREBASE_SERVICE_ACCOUNT_BASE64 sem terminal.
Não suba o arquivo serviceAccountKey.json no GitHub. Selecione ele apenas nesta página.

1. Baixar chave no Firebase

Firebase Console → Configurações do projeto → Contas de serviço → Gerar nova chave privada.

Depois renomeie para serviceAccountKey.json.

Esse JSON é privado. Guarde apenas no seu computador.

2. Converter para Base64

Base64 copiado!

3. Colar na Vercel

Vercel → Projeto gestorflex-zeta → Settings → Environment Variables.

Name: FIREBASE_SERVICE_ACCOUNT_BASE64
Value: cole o Base64
Environment: Production

Abrir Vercel
Nome copiado!

4. Testar

Depois de salvar a variável, faça Redeploy e teste:

Abrir teste
Link copiado!
Se retornar ok: true, está pronto.
